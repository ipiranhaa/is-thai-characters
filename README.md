# IS THAI CHARACTERS

Thai string validator

## Installation

**NPM**

`npm install is-thai-characters`

**Yarn**

`yarn add is-thai-characters`

## Usage

```
import isThaiCharacters from 'is-thai-characters'

isThaiCharacters('สวัสดี')    => true
isThaiCharacters('Hello')   => false
```
